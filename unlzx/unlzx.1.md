# NAME
unlzx - uncompress .lzx archive made on Amiga

# SYNOPSIS
Usage: unlzx [-v][-x][-c] [archive...]

# DESCRIPTION
LZX is the name of an LZ77 family compression algorithm
popular on the Amiga family computer
unlzx uncompress those archive

# OPTIONS
\-c
:   extract/list from stdin

\-v
:   list archive(s)

\-x
:   extract (default)

# SEE ALSO
`unlzx`(1)
