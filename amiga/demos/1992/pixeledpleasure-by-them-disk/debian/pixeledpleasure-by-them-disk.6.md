# NAME
pixeledpleasure-by-them-disk - launcher for Pixeled Pleasure

# SYNOPSIS
pixeledpleasure-by-them-disk [*options*]

# DESCRIPTION
pixeledpleasure-by-them-disk - is the launcher for Pixeled Pleasure that use fs-uae in back-end.
Demo Released in 1991

# OPTIONS
\--help
:   Display help for the command

\--fullscreen
:   Start in fullscreen

\--version
:   Display version and author information

# NOTE
An original Kickstart is **not** necessary because it uses the builtin Aros Kickstart.
