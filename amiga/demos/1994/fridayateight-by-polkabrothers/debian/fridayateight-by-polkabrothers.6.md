# NAME
fridayateight-by-polkabrothers - launcher for Friday at Eight

# SYNOPSIS
fridayateight-by-polkabrothers [*options*]

# DESCRIPTION
fridayateight-by-polkabrothers - is the launcher for Friday at Eight that use fs-uae in back-end.  
Demo ranked 3rd at The Gathering 1994

# OPTIONS
\--help
:   Display help for the command

\--fullscreen
:   Start in fullscreen

\--version
:   Display version and author information

# NOTE
An original Kickstart is **not** necessary because it uses the builtin Aros Kickstart.

`fs-uae`(1)
