# NAME
deep-by-parallaxandcncd - launcher for Deep

# SYNOPSIS
deep-by-parallaxandcncd [*options*]

# DESCRIPTION
deep-by-parallaxandcncd - is the launcher for Deep that use fs-uae in back-end.  
Demo ranked 1st at The Gathering 1995

# OPTIONS
\--help
:   Display help for the command

\--fullscreen
:   Start in fullscreen

\--version
:   Display version and author information

# NOTE
An original Kickstart is **not** necessary because it uses the builtin Aros Kickstart.

# SEE ALSO
`fs-uae`(1)
