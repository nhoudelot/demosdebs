# NAME
getrom - the command to run getrom.

# SYNOPSIS

# DESCRIPTION
getrom is a program released by Sylvain Huet in 1996.

Recuperation de la rom du to7 a partir d'un fichier .k7
La rom a du etre sauvegardee par : SAVEM "ROM",&HE800,&HFFFF,0
