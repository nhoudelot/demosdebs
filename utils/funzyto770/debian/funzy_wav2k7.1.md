# NAME
wav2k7 - the command to run wav2k7.

# SYNOPSIS
wav2k7 file.wav

# DESCRIPTION
wav2k7 is a program released by Sylvain Huet in 1996.
conversion .wav 44,1KHz -> .k7 to7

version 2.1

# OPTIONS
there is no options to this program
