# NAME
x220-by-hackers - the command to run X220.

# SYNOPSIS
x220-by-hackers [*options*]

# DESCRIPTION
X220 is a demonstration released by Hackers in 2017.

# OPTIONS
\--windowed
:   Start the demo in window mode

\--help
:   Display help for the command
