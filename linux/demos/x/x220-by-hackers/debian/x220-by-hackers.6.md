# NAME
x220-by-hackers - the command to run X220.

# SYNOPSIS
x220-by-hackers [*options*]

# DESCRIPTION
X220 is a demo released by Hackers in may 2017.
Ended 2nd at Skrolli Party 2017

ash checksum [Code, design]
