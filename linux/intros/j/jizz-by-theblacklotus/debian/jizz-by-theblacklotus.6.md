# NAME
**jizz-by-theblacklotus** - launch the Jizz 64 KB intro by The Black Lotus

# SYNOPSIS
**jizz-by-theblacklotus** [**--help**]

# DESCRIPTION
**Jizz** is a 64 KB intro created by *The Black Lotus* and entered in the
intro competition at the Wired'97 demoparty in 1997.

The demo requires a 486DX processor and a 320×200 display in 15 or 16 bits
per pixel. For optimal performance, an iP166 processor or faster is
recommended, along with a GUS sound card with 1 MB of RAM and VBE 2.0
support with linear framebuffer (LFB) access.

This Debian/Ubuntu package runs the demo using the **ixalance**(1) emulator.
