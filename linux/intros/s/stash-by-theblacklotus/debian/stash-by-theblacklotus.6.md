# NAME
**stash-by-theblacklotus** - run the Stash 64Kb intro by The Black Lotus

# SYNOPSIS
**stash-by-theblacklotus** [**--help**]

# DESCRIPTION
**Stash** is a 64\ Kb intro released by **The Black Lotus** at a demoparty in 1997.
It features 3.5\ MB of full-colour graphics and 4\ MB of music sampled at
16-bit/44\ kHz, all packed into a 64\ Kb executable.

The production was created by probe, jace, nix, balance, danny, and
sick1/trp.
Running the program requires a Pentium-class CPU or better, a VGA card, and
a Gravis UltraSound with at least 1\ MB of sample RAM.

# OPTIONS
**--help**
:   Display usage information and exit.
