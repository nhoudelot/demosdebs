# NAME
emodefect-by-fit - is the command to run Emo Defect 

# SYNOPSIS
emodefect-by-fit [*options*]

# DESCRIPTION
Emo Defect  is a 64 ko intro released by Fit in august 2006
Ended 5th at Assembly 2006

Marq [Code, Graphics]
roz [Music]

# OPTIONS
\-w
:	run this intro in a window

# KEYS
F
:	toggle fullscreen

ESC or Q
:	quit
