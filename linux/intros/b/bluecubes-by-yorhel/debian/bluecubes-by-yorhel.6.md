# NAME
bluecubes-by-yorhel - is the command to run Blue Cubes 

# SYNOPSIS
bluecubes-by-yorhel [*options*]

# DESCRIPTION
Blue Cubes  is a 64k intro released by YorHel released in august 2006
It ended 4th combined 64k at  Evoke 2006

# OPTIONS
\-1
:   for 800x600

\-2
:   for 1024x768

\-3
:   for 1280x960

\-w
:   for window mode

# KEYS
ESC / Q
:	to quit
F
:	toggle fullscreen
