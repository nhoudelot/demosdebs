# NAME

funzyto770 - Thomson TO7 and TO7-70 home computer emulator

# SYNOPSIS

**funzyto770** [**-k** *keyfile*] [**-gray**] [**-mono**]

# DESCRIPTION

**funzyto770** emulates the Thomson TO7 and TO7-70, French 8-bit home
computers based on the Motorola 6809E processor. The emulator requires an
X11 display and emulates the following hardware:

- Motorola 6809E CPU
- optical pen (light pen)
- joysticks (2)
- cassette tape reader (.k7 format)
- removable memo7 ROM cartridges (16 KB each)

On startup, the emulator loads the TO7 high ROM from the file **romto770**
and the BASIC cartridge from **memo7/basic**, both expected in the current
working directory. It then drops into the interactive monitor.

## Monitor

The monitor is a text-mode interface that controls the emulator. The
following commands are available:

**go**
:   Start or resume emulation.

**reset**
:   Simulate pressing the TO7 reset button.

**k7** *filename*
:   Load a new cassette tape file (.k7 format).

**seek** *hex*
:   Set the cassette tape position (hexadecimal value).

**load** *filename*
:   Load a memo7 ROM cartridge image.

**speed** *hex*
:   Adjust the emulation slow-down loop counter. Use 0 for full speed.

**dir**
:   List files in the current directory.

**cd** *directory*
:   Change the current directory.

**pwd**
:   Print the current directory.

**q**
:   Quit the emulator.

## Mouse controls during emulation

**Right button**
:   Suspend emulation and return to the monitor.

**Middle button**
:   Move the light pen across the screen (held down while moving).

**Left button**
:   Press the light pen tip (trigger).

# OPTIONS

**-k** *keyfile*
:   Keyboard layout file to use. Defaults to **keyto7** in the current
    directory. This file is generated by **funzy_enterkey**(1).

**-gray**
:   Use a greyscale palette (for greyscale displays).

**-mono**
:   Use strictly monochrome rendering (for monochrome displays).

# FILES

*keyto7*
:   Default keyboard layout file, generated by **funzy_enterkey**(1).

*romto770*
:   TO7 high ROM image (6144 bytes). Required. Place in the working
    directory. Obtain with **funzy_getrom**(1).

*memo7/basic*
:   BASIC cartridge image (16384 bytes). Loaded automatically on startup.
    Obtain with **funzy_getmemo7**(1).

*memo7/*
:   Directory holding additional memo7 cartridge images.

# EXAMPLES

Run the emulator with the default keyboard layout:

    funzyto770

Run the emulator with a custom keyboard layout:

    funzyto770 -k mykeymap

Once in the monitor, start emulation:

    ok
    0000 > go

Load a program from cassette and run it:

    ok
    0000 > k7 myprog.k7
    ok
    0000 > go
