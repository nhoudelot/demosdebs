# NAME
bluecubes-by-yorhel - is the command to run Blue Cubes 

# SYNOPSIS
bluecubes-by-yorhel [*options*]

# DESCRIPTION
Blue Cubes  is a demonstration released by YorHel
Demo by YorHel, released in 2006

# OPTIONS

\-1
:   for 800x600

\-2
:   for 1024x768

\-3
:   for 1280x960

\-w
:   for window mode
