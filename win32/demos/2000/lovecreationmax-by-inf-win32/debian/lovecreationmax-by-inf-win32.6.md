# NAME
lovecreationmax-by-inf-win32 - launcher for Love Creation Max

# SYNOPSIS
lovecreationmax-by-inf-win32 [*options*]

# DESCRIPTION
lovecreationmax-by-inf-win32 - is the launcher for Love Creation Max that call PlayOnLinux in backend.
Demo ranked 1st at The Party 2000

# OPTIONS
\--help
:   Display help for the command

\--version
:   Display version and author information

# ENVIRONMENT VARIABLES
\ APP_RES [widthxheight]
:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 lovecreationmax-by-inf-win32
