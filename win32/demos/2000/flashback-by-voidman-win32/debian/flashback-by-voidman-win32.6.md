# NAME
flashback-by-voidman-win32 - launcher for Flashback

# SYNOPSIS
flashback-by-voidman-win32 [*options*]

# DESCRIPTION
flashback-by-voidman-win32 - is the launcher for Flashback that call PlayOnLinux in backend.
Demo ranked 8th at Dialogos 2000

# OPTIONS
\--help
:   Display help for the command

\--version
:   Display version and author information


# ENVIRONMENT VARIABLE
\ APP_RES [widthxheight]
:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 flashback-by-voidman-win32
