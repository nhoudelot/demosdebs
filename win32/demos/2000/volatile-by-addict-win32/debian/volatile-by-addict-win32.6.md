# NAME
volatile-by-addict-win32 - launcher for Volatile

# SYNOPSIS
volatile-by-addict-win32 [*options*]

# DESCRIPTION
volatile-by-addict-win32 - is the launcher for Volatile that call PlayOnLinux in backend.
Demo ranked 3rd at Satellite 2000

# OPTIONS
\--help
:   Display help for the command

\--version
:   Display version and author information

# ENVIRONMENT VARIABLEs
\ APP_RES [widthxheight]
:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 volatile-by-addict-win32
