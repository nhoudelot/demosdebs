# NAME
haugyjobb-by-astral-win32 - launcher for Hau(gy)jobb

# SYNOPSIS
haugyjobb-by-astral-win32 [*options*]

# DESCRIPTION
haugyjobb-by-astral-win32 - is the launcher for Hau(gy)jobb that call PlayOnLinux in backend.
Demo ranked 8th at Assembly 2000
