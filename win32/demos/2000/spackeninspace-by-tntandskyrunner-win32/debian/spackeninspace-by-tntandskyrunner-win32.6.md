# NAME
spackeninspace-by-tntandskyrunner-win32 - launcher for Spacken in Space

# SYNOPSIS
spackeninspace-by-tntandskyrunner-win32 [*options*]

# DESCRIPTION
spackeninspace-by-tntandskyrunner-win32 - is the launcher for Spacken in Space that call PlayOnLinux in backend.
Demo ranked 9th at Dialogs 2000
