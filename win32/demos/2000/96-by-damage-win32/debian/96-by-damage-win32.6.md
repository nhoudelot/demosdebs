# NAME
96-by-damage-win32 - launcher for 96

# SYNOPSIS
96-by-damage-win32 [*options*]

# DESCRIPTION
96-by-damage-win32 - is the launcher for 96 that call PlayOnLinux in backend.
Demo ranked 10th at Mekka and Symposium 2000

# OPTIONS
\--help
:   Display help for the command

\--version
:   Display version and author information

# ENVIRONMENT VARIABLE
\ APP_RES [widthxheight]
:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 96-by-damage-win32
