# NAME
further-by-moppiproductions-win32 - launcher for Further

# SYNOPSIS
further-by-moppiproductions-win32 [*options*]

# DESCRIPTION
further-by-moppiproductions-win32 - is the launcher for Further that call PlayOnLinux in backend.
Demo seen at Assembly 2000

# OPTIONS
\--help
:   Display help for the command

\--version
:   Display version and author information

# ENVIRONMENT VARIABLES
\ APP_RES [widthxheight]
:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 further-by-moppiproductions-win32
