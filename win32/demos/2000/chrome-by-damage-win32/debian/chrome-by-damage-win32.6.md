# NAME
chrome-by-damage-win32 - launcher for Chrome

# SYNOPSIS
chrome-by-damage-win32 [*options*]

# DESCRIPTION
chrome-by-damage-win32 - is the launcher for Chrome that call PlayOnLinux in backend.
Demo ranked 2rd at The Party 2000

# OPTIONS
\--help
:   Display help for the command

\--version
:   Display version and author information

# ENVIRONMENT VARIABLE
\ APP_RES [widthxheight]
:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 chrome-by-damage-win32
