# NAME
fairplaytothequeen-by-candela-win32 - the command to run Fair play to the queen.

# SYNOPSIS
fairplaytothequeen-by-candela-win32 [*options*]

# DESCRIPTION
Fair play to the queen is a demonstration released by Candela in 2005.
fairplaytothequeen-by-candela-win32 is the launcher that use Wine & PlayOnLinux to run it.

# OPTIONS
\--setdesktop
:   Set to desktop mode and run (can go in fullscreen)

\--help
:   Display help for the command

# ENVIRONMENT VARIABLEs
\ APP_RES [widthxheight]
:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 fairplaytothequeen-by-candela-win32
