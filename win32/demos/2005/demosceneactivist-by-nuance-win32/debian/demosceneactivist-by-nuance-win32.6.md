# NAME
demosceneactivist-by-nuance-win32 - launcher for Demoscene Activist

# SYNOPSIS
demosceneactivist-by-nuance-win32 [*options*]

# DESCRIPTION
demosceneactivist-by-nuance-win32 - is the launcher for Demoscene Activist that call PlayOnLinux in backend.
demo ranked 6th at Breakpoint 2005

# OPTIONS
\--help
:   Display help for the command

\--version
:   Display version and author information

# ENVIRONMENT VARIABLEs
\ APP_RES [widthxheight]
:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 demosceneactivist-by-nuance-win32
