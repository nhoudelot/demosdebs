# NAME
sts04instantzen-by-synesthetics-win32 - launcher for STS-04: Instant Zen

# SYNOPSIS
sts04instantzen-by-synesthetics-win32 [*options*]

# DESCRIPTION
sts04instantzen-by-synesthetics-win32 - is the launcher for STS-04: Instant Zen that call PlayOnLinux in backend.
demo ranked 2nd at Assembly 2005

\--help
:   Display help for the command

\--version
:   Display version and author information

# ENVIRONMENT VARIABLEs

:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 sts04instantzen-by-synesthetics-win32
