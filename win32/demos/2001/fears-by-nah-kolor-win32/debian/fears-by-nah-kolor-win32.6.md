# NAME
fears-by-nah-kolor-win32 - launcher for Fears

# SYNOPSIS
fears-by-nah-kolor-win32 [*options*]

# DESCRIPTION
fears-by-nah-kolor-win32 - is the launcher for Fears that call PlayOnLinux in backend.
Demo ranked 1st at Abstract 2001

# OPTIONS
\--help
:   Display help for the command

\--version
:   Display version and author information

# ENVIRONMENT VARIABLEs
\ APP_RES [widthxheight]
:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 fears-by-nah-kolor-win32
