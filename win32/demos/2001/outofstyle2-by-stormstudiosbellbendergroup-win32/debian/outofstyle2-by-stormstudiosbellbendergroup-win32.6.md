# NAME
outofstyle2-by-stormstudiosbellbendergroup-win32 - launcher for Out Of Style 2

# SYNOPSIS
outofstyle2-by-stormstudiosbellbendergroup-win32 [*options*]

# DESCRIPTION
outofstyle2-by-stormstudiosbellbendergroup-win32 - is the launcher for Out Of Style 2 that call PlayOnLinux in backend.
Demo ranked 15th at Assembly 2001

# OPTIONS
\--help
:   Display help for the command

\--version
:   Display version and author information

# ENVIRONMENT VARIABLES
\ APP_RES [widthxheight]
:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 outofstyle2-by-stormstudiosbellbendergroup-win32
