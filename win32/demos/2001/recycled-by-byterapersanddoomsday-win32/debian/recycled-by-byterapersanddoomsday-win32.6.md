# NAME
recycled-by-byterapersanddoomsday-win32 - launcher for Recycled

# SYNOPSIS
recycled-by-byterapersanddoomsday-win32 [*options*]

# DESCRIPTION
recycled-by-byterapersanddoomsday-win32 - is the launcher for Recycled that call PlayOnLinux in backend.
Demo ranked 3rd at Assembly 2001

# OPTIONS
\--help
:   Display help for the command

\--version
:   Display version and author information

# ENVIRONMENT VARIABLES
\ APP_RES [widthxheight]
:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 recycled-by-byterapersanddoomsday-win32
