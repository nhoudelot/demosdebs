# NAME
stigma-by-emotive-win32 - launcher for stigma

# SYNOPSIS
stigma-by-emotive-win32 [*options*]

# DESCRIPTION
stigma-by-emotive-win32 - is the launcher for stigma that call PlayOnLinux in backend.
demo ranked 1st at Ambience 2001

# OPTIONS
\--help
:   Display help for the command

\--version
:   Display version and author information

# ENVIRONMENT VARIABLES
\ APP_RES [widthxheight]
:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 stigma-by-emotive-win32
