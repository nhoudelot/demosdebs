# NAME
gasoline2-by-recreation-win32 - launcher for Gasoline 2

# SYNOPSIS
gasoline2-by-recreation-win32 [*options*]

# DESCRIPTION
gasoline2-by-recreation-win32 - is the launcher for Gasoline 2 that call PlayOnLinux in backend.
Demo ranked #13 at Assembly 2001

# OPTIONS
\--help
:   Display help for the command

\--version
:   Display version and author information

# ENVIRONMENT VARIABLEs
\ APP_RES [widthxheight]
:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 gasoline2-by-recreation-win32
