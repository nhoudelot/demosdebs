# NAME
obe-byvirtualdreamsfairlightsumealabs-win32 - launcher for OBE

# SYNOPSIS
obe-byvirtualdreamsfairlightsumealabs-win32 [*options*]

# DESCRIPTION
obe-byvirtualdreamsfairlightsumealabs-win32 - is the launcher for OBE that call PlayOnLinux in backend.
Demo ranked 10th at Assembly 2001

:   Display version and author information

# ENVIRONMENT VARIABLES
