# NAME
legomania-by-doomsday-win32 - launcher for Legomania

# SYNOPSIS
legomania-by-doomsday-win32 [*options*]

# DESCRIPTION
legomania-by-doomsday-win32 - is the launcher for Legomania that call PlayOnLinux in backend.
demo ranked 1st at Asembly 2003

# OPTIONS
\--help
:   Display help for the command

\--version
:   Display version and author information

# ENVIRONMENT VARIABLEs
\ APP_RES [widthxheight]
:	start the app in a widthxheight virtual desktop  
	ex. APP_RES=800x600 legomania-by-doomsday-win32
